## Keybinds
WSUPER + R = Rofi Launcher
SUPER + W = Kill/Close Window
SUPER + Ctrl + R = Restart Qtile
SUPER + U = Shrink Window Size
SUPER + Y = Increase Window Size
SUPER + M = Maximize Window
F11 = Toggle Fullscreen
SUPER + Ctrl + S = Shutdown Qtile
Print = Screenshot
